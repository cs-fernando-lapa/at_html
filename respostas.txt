1) Quando é recomendando a criação de um site separado para mobile? Seria
este o caso deste site da GK Fashion?

É recomendado um segundo site, quando a performance deve ser levada em conta, e caso o site tenha sido desenvolvido primeiramente só pensando em Desktop.
Também é correto afirmar que a velocidade de leitura de uma site responsivo é muito maior, e se levado em consideração a velocidade na internet no Brasil, como a 3G/4G todos os sites deveriam ser 
separados.

No caso da GK Fashion, por se tratar de um site de vendas, a lentidão pode acarretar na desistência de um cliente em comprar o produto, nesse caso acredito que seria mais eficiente
o desenvolvimento de um site separado para mobile, apenas acrescentando o m.gkfashion.com.br na regra do DNS para redirecionamento.


2) No caso de um CSS específico para print, como ele deve ser inserido?

Criando um css e linkando como é feito o link das media queries. 
No css utiliza o @media print {
   …
}
E no index.html utiliza:
    
<link rel="stylesheet" media=“print” href="css/mobile.css">
